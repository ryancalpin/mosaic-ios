# TUI Mode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** When a command enters alternate screen mode (vim, htop, Claude Code, OpenCode, Hermes), switch SessionView from the native rendering layout to a full-screen SwiftTerm view with raw keyboard passthrough. Switch back when alternate screen exits.

**Architecture:** `Session` gets a `@Published var isTUIMode: Bool`. `handleOutput(_:)` watches for `\033[?1049h` (enter alternate screen) and `\033[?1049l` (exit). `SessionView` switches layout based on this flag — TUI mode shows the `TerminalViewBridge` at full opacity with hit-testing enabled and first responder, replacing the ScrollView+OutputBlocks+SmartInputBar stack with a minimal ESC/Ctrl-C toolbar. `BreadcrumbBar` shows a "TUI" pill badge while active.

**Tech Stack:** Swift, SwiftUI, SwiftTerm (`TerminalViewBridge`), `@Published`, UIKit first responder

### Task 3: TUIControlBar — ESC and Ctrl-C Passthrough Buttons

**Files:**

- [ ] **Step 1: Create the file**

```swift
// Sources/Mosaic/UI/Session/TUIControlBar.swift
import SwiftUI

// Minimal toolbar shown at the bottom of the screen in TUI mode.
// Provides hardware-keyboard-less users access to ESC and Ctrl-C.
struct TUIControlBar: View {
    let onSendBytes: (Data) -> Void

    var body: some View {
        HStack(spacing: 0) {
            TUIKey(label: "ESC", detail: "⎋") {
                onSendBytes(Data([0x1B]))
            }

            Divider()
                .frame(height: 20)
                .overlay(Color.mosaicBorder)

            TUIKey(label: "Ctrl-C", detail: "^C") {
                onSendBytes(Data([0x03]))
            }

            Divider()
                .frame(height: 20)
                .overlay(Color.mosaicBorder)

            TUIKey(label: "Ctrl-Z", detail: "^Z") {
                onSendBytes(Data([0x1A]))
            }

            Spacer()

            Text("TUI MODE")
                .font(.custom("JetBrains Mono", size: 8).weight(.bold))
                .kerning(0.4)
                .foregroundColor(.mosaicYellow)
                .padding(.trailing, 16)
        }
        .frame(height: 44)
        .background(Color.mosaicSurface1)
        .overlay(
            Rectangle().fill(Color.mosaicBorder).frame(height: 0.5),
            alignment: .top
        )
    }
}

private struct TUIKey: View {
    let label: String
    let detail: String
    let action: () -> Void

    var body: some View {
        Button(action: {
            UIImpactFeedbackGenerator(style: .light).impactOccurred()
            action()
        }) {
            VStack(spacing: 1) {
                Text(detail)
                    .font(.system(size: 14, weight: .medium, design: .monospaced))
                    .foregroundColor(.mosaicTextPri)
                Text(label)
                    .font(.system(size: 9))
                    .foregroundColor(.mosaicTextSec)
            }
            .frame(width: 64, height: 44)
            .contentShape(Rectangle())
        }
        .buttonStyle(.plain)
    }
}
```

- [ ] **Step 2: Conditional layout**

Replace the existing body with a conditional that switches between two layouts. The current body is roughly:

```swift
ZStack {
    TerminalViewBridge(...)
        .opacity(0)
        .allowsHitTesting(false)
    
    ScrollView { /* OutputBlocks */ }
    
    ApprovalCardView(...)
}
// + SmartInputBar below
```

Replace with:

```swift
var body: some View {
    VStack(spacing: 0) {
        BreadcrumbBar(
            username: session.username,
            hostname: session.hostname,
            directory: session.currentDirectory,
            branch: session.branch,
            ahead: session.ahead,
            isTUIMode: session.isTUIMode   // pass through
        )

        if session.isTUIMode {
            // TUI layout: full-screen terminal, no scroll
            TerminalViewBridge(
                session: session,
                isTUIMode: true
            )
            .frame(maxWidth: .infinity, maxHeight: .infinity)

            TUIControlBar { data in
                session.connection.send(data)
            }
        } else {
            // Normal layout: hidden terminal + output blocks + input bar
            ZStack {
                TerminalViewBridge(
                    session: session,
                    isTUIMode: false
                )
                .opacity(0)
                .allowsHitTesting(false)

                ScrollViewReader { proxy in
                    ScrollView {
                        LazyVStack(spacing: 0) {
                            ForEach(session.blocks) { block in
                                OutputBlockView(block: block)
                            }
                        }
                    }
                    .onChange(of: session.blocks.count) { _, _ in
                        if let last = session.blocks.last {
                            proxy.scrollTo(last.id, anchor: .bottom)
                        }
                    }
                }

                ApprovalCardView(session: session)
            }

            SmartInputBar(session: session)
        }
    }
    .background(Color.mosaicBg)
    .animation(.easeInOut(duration: 0.2), value: session.isTUIMode)
}
```

Adjust property names to match the actual Session API (e.g., `session.username`, `session.currentDirectory`) as seen in the existing code.
